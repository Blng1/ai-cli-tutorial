01 MCP 是什么

Model Context Protocol — AI 与外部世界之间的标准化桥梁。

核心定义

MCP (Model Context Protocol) 是 Anthropic 于 2024 年发起的开放协议，定义了 AI 模型与外部工具/数据源之间的标准通信方式。现已被 OpenAI、Google 等主要厂商采纳，成为 AI 工具扩展的事实标准。

// .claude/settings.json
{
  "mcpServers": {
    "github": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"],

        "GITHUB_TOKEN": "your-token"
      }
    },

      "command": "npx",

    }
  }

04 MCP 使用建议

合理配置 MCP 是高效使用 AI 工具的关键，避免贪多嚼不烂。

MCP 配置最佳实践

1) 从核心 3 个开始：GitHub + 一个数据库 + 一个搜索工具，覆盖 80% 日常场景；
2) 注意安全：不要在 MCP 配置中暴露生产环境凭证，使用环境变量或密钥管理服务；
3) 按需添加：不要贪多，每个 MCP Server 都消耗 context window，过多会降低 AI 推理质量；
4) 团队统一：通过项目配置文件统一 MCP 配置，确保团队一致性，避免"在我机器上能跑"的问题。
